# Homepage Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Restyle the localized Hàm Ninh homepage to closely match the approved reference without changing the shared header, backend, routes, translations, or application behavior.

**Architecture:** Keep `frontend/src/app/[locale]/page.tsx` as the server-side locale and translation boundary. Implement the visual redesign inside the existing `TourismHomepage` presentational component, retaining its typed content contract, locale-aware links, local image asset, and semantic section IDs. Add focused source-contract and Playwright visual checks to lock the approved page structure and responsive behavior.

**Tech Stack:** Next.js 16 App Router, React 19, TypeScript 6, Tailwind CSS 4, next-intl, next/image, Lucide React, Node test scripts, Playwright.

### Task 5: Add Responsive Browser Verification

**Files:**

- [ ] **Step 1: Write the Playwright verification script**

```js
import assert from "node:assert/strict";
import { mkdir } from "node:fs/promises";
import { chromium } from "@playwright/test";

const baseURL = process.env.BASE_URL ?? "http://127.0.0.1:3500";
const executablePath = process.env.PLAYWRIGHT_CHROMIUM_EXECUTABLE_PATH;
const outputDir = new URL("../test-results/homepage-ui/", import.meta.url);

await mkdir(outputDir, { recursive: true });

const browser = await chromium.launch({
  headless: true,
  ...(executablePath ? { executablePath } : {}),
});

const checks = [
  { name: "desktop-vi", path: "/vi", viewport: { width: 1440, height: 1000 } },
  { name: "mobile-vi", path: "/vi", viewport: { width: 375, height: 812 } },
  { name: "desktop-en", path: "/en", viewport: { width: 1440, height: 1000 } },
];

for (const check of checks) {
  const page = await browser.newPage({ viewport: check.viewport });
  const errors = [];
  page.on("pageerror", (error) => errors.push(error.message));

  await page.goto(`${baseURL}${check.path}`, { waitUntil: "networkidle" });

  for (const selector of [
    "#homepage-hero",
    "#homepage-stats",
    "#homepage-benefits",
    "#homepage-steps",
    "#homepage-cta",
    "#homepage-footer",
  ]) {
    await page.locator(selector).waitFor({ state: "visible" });
  }

  const overflow = await page.evaluate(
    () => document.documentElement.scrollWidth > document.documentElement.clientWidth,
  );

  assert.equal(overflow, false, `${check.name} should not overflow horizontally`);
  assert.deepEqual(errors, [], `${check.name} should not emit page errors`);

  await page.screenshot({
    path: new URL(`${check.name}.png`, outputDir),
    fullPage: true,
  });
  await page.close();
}

await browser.close();
console.log("S12 homepage visual check passed.");
```
